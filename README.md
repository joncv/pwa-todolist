# pwa-todolist

> 这是一个支持PWA的TodoList，使用Lavas进行构建

## npm 相关命令

``` bash
# 安装工程依赖
npm install

# 更新工程依赖
npm update

# 在 localhost:8082 上启动带有热更新机制的开发服务器
npm run dev

# 构建线上生产环境产物
npm run build

```
## 基本功能
- 登录注册
- 重要紧急程度分类（参考"高效Todo"App）
- 支持Markdown

## 基本手势操作
1. 点击：查看详情
2. 长按：删除
3. 右划（仅在“未完成”页面有效）：设置为“已完成”
4. 左划（仅在“已完成”页面有效）：设置为“未完成”
